About Me

Himanshu Mishra

I'm an MSc IT (AI) student with a BSc IT background, passionate about using Python and AI to solve real-world problems. I'm currently building expertise in Machine Learning, Data Analysis, and Deep Learning, with hands-on practice in tools like pandas, scikit-learn, and TensorFlow.

In my final year of BSc, I developed a complete web application titled "Transport Module" using ASP.NET Core MVC and SQL Server, automating transport and logistics workflows. This experience gave me strong foundations in backend development, database design, and clean coding practices.

I'm also exploring Generative AI with AWS, building projects that combine AI models with real-world applications. My learning journey includes working on DSA, RESTful APIs, and deploying models via Flask/Streamlit to the cloud.

I'm committed to continuous learning and open to roles in AI/ML engineering, data science, Python development, or backend engineering. Let's connect if you're building something innovative with technology.
